# 项目相关说明文档请查看 docz 文件夹

项目结构
@types: 全局 ts/第三方库 ts 补充
assets: 静态资源
components: 基础组件
containers: 业务代码
containers/shared: 业务代码组件
containers/view: 业务页面

feature:

-   [x] react
-   [x] typescript
-   [x] antd
-   [x] styled-components (css in js)
-   [x] husky + staged-lint (提交前检测)
-   [x] React Router v5 (路由)
-   [x] constate (状态管理)
-   [x] qs (url 处理)
-   [x] moments (时间处理)
-   [x] lodash-es (工具函数)
-   [x] ahooks（常用 hooks 的封装）
-   [ ] svgo (svg 图标处理)
